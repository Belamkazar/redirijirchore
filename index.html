Formulario de contacto

En este ejemplo, se utiliza el objeto navigator.userAgent para verificar si el navegador es Google Chrome. Si es así, se utiliza window.open() para abrir una nueva ventana o pestaña con la dirección especificada. De lo contrario, se muestra la ventana emergente como se describió anteriormente. Ten en cuenta que esta es solo una forma de detectar el navegador y redirigir al usuario a otra dirección en caso de que estén utilizando Google Chrome. Puedes adaptar el código según tus necesidades específicas.
